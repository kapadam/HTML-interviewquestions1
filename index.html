8. Difference between HTML Tags and Elements?

HTML tags- Tags are the starting and ending part of an HTML element. They begin with “<” symbol and end with “>”. Whatever written inside < and > are called tags. For instance: -- <b></b>.

HTML elements – Elements enclose the contents in between the tags. They consist of some kind of structure. It generally consists of a start tag, content and end tag. For instance: -- <b>Content</b>.

9. Explain <br/> tag?

The <br> tag in HTML document is used to create a line break in a text. It is an empty tag which means it does not need end tag like </br>.

It is generally used in poem and address where the division of line is necessary.

10. Explain how many different types of lists we have in HTML?

There are two types of lists in HTML:

  1. Ordered List
  2. Unordered List

1. Ordered List:

  • Numbers:
    1. Item 1
    2. Item 2
  • Lowercase Roman numbers:
    1. Item 1
    2. Item 2
  • Uppercase Roman numbers:
    1. Item 1
    2. Item 2
  • Lowercase Alphabets:
    1. Item 1
    2. Item 2
  • Uppercase Alphabets:
    1. Item 1
    2. Item 2

2. Unordered List:

  • Default:
    • Item 1
    • Item 2
  • None:
    • Item 1
    • Item 2
  • Circle:
    • Item 1
    • Item 2
  • Disc:
    • Item 1
    • Item 2
  • Square:
    • Item 1
    • Item 2

11. What are Semantic Tags?

A semantic tag in HTML is a tag whose name clearly describes its purpose and content.

HTML5 introduced several new semantic tags.

These include <main>, <aside>, <section>, <article> and others.

12. Write a code to create an ordered list with roman numbers?

Code:
    <ol type="I">
        <li>Apple</li>
        <li>Orange</li>
        <li>Banana</li>
    </ol>
Output:
  1. Apple
  2. Orange
  3. Banana

12. Write a code to create an unordered list with circle?

Code:
    <ul type="circle">
        <li>Apple</li>
        <li>Orange</li>
        <li>Banana</li>
    </ul>
Output:
  • Apple
  • Orange
  • Banana
